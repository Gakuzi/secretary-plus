Мастер Настройки Секретарь+

Следуйте шагам, чтобы подключить и настроить вашего ассистента.

1

Подключение к Supabase и Аутентификация

Введите данные вашего проекта Supabase, чтобы включить облачную синхронизацию и сохранить настройки. После этого войдите через Google.

2

Настройка Базы Данных

Теперь необходимо настроить таблицы в вашей базе данных. Рекомендуется использовать автоматический метод.

Этот метод использует официальный API Supabase. Он на 100% надежный и безопасный, так как обходит ограничения браузера.

Шаг 2.1: Сгенерируйте токен доступа

  1. Откройте страницу токенов доступа.
  2. Нажмите "Generate New Token", дайте имя и скопируйте его.

Шаг 2.2: Выполните команду

Скопируйте сгенерированную команду и выполните её в вашем терминале. Ваш ID проекта уже подставлен автоматически.

Введите токен доступа выше...
  1. В меню вашего проекта выберите SQL Editor, затем "+ New query".
  2. Скопируйте и вставьте весь SQL-скрипт ниже в редактор.
  3. Нажмите "RUN".
